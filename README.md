# FIFA World Cup 2026 Typer

Statyczna aplikacja do typowania wyników FIFA World Cup 2026.

## Struktura

- `index.html` - lekki plik startowy
- `styles/main.css` - wszystkie style aplikacji
- `src/app.js` - logika aplikacji React/Firebase oraz skrypty pomocnicze
- `work/` - lokalne kopie zapasowe i pliki robocze, nie trzeba ich publikować

## Uruchomienie lokalne

Możesz otworzyć `index.html` bezpośrednio w przeglądarce.

## GitHub Pages

Repozytorium może być publikowane jako statyczna strona przez GitHub Pages. Ustaw źródło Pages na gałąź z plikiem `index.html`.

## Uwaga o Firebase

Konfiguracja Firebase jest widoczna w kodzie frontendu. To normalne dla aplikacji webowych, ale bezpieczeństwo musi być ustawione w Firebase Realtime Database Rules.
